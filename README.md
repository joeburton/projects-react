# Node, Express, Handlebars, MongoDB, React & Redux CRUD Apllication

```
// Install MongoDB locally.
$ cd /usr/local/mongodb

// Step 2: start mongoDB
$ ./bin/mongod

// Step 3: start mongo shell
$ ./bin/mongo

// Step 4: select/ create database
$ use DB_NAME
```

```
start the App
$ npm install
$ nodemon
```
